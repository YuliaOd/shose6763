Велосипеди

Підберається в залежності від того де планується використування. Це може бути велосипед для швидкого приїзду на роботу, для походу, кататися по парку та відпочивати. Чи можливо для поїздок з друзями на пікнік.

    Їзда за містом по бездоріжжю.

    вилка амортизуюча

    Передачі великий діапазон

    Великий діаметр колес, для того щоб легше проходити нерівності. Амортизаційна вилка, для того щоб їздити по камінню та кочкам. Великий діапазон передач для того щоб було легше підніматися до гори.

    Міський

    Для їзди в місті та по асфальту.

    рама стальна або алюміньова
    вилка жортска, без амортизаторів
    гальма v-brake
    Передачі відсутні або небагато

    Широке комфортне сідло. Часто доповнюється корзинкою або багажником. Широке та високе кермо для прямої посадки.

    Електричний

    Бувають для міста та для бездоріжжя.

    рама стальна або алюміньова
    вилка жортска або з амортизатором
    гальма v-brake або дискові

    Двигун надає підтримку до 25 км/ч. Вага більш висока. Відрізняються часом заряду та дальністтю поїздки.

    Складний

    Для поїздок у місті

    рама стальна або алюміньова
    вилка жортска
    гальма v-brake
    Передачі є невелика кількість

    Займають меньше місця, простіше перевозити. Чим швижше і простіше складаєтся ти дорожче ціна.

    Шосейний

    Для швидкостної їзди по дуже рівної поверхні

    рама алюміньова
    вилка жортска
    гальма v-brake
    Передачі велика кількість

    Спортивна посадка та вигнуте кермо. Дуже вузькі колеса с гладкими покришками які знижують тертя.

    Передачі відсутні

    Тільки для "флета" та "стріта". Кермо крутиться на 360. Спеціальні труби на осях "пеги". Пеги потрібні для виконання різноманітних трюків.

    Швидкості

    Швидкості дають змогу економити сили при їзді по місту і ефективно розгонятися на рівній поверхні. Включає в себе: задні зірки та перемикач, монетки на кермі (з їх допомогаю перемикаємо швидкості), передні зірки та перемикачі. Ціль системи - перекидати ланцюг з великих зірок на маленькі і навпаки. Система = шатуни + зірки.

    Правила перемикання швидкостей

  • перемикаємо тільки в процесі руху велосипеду;
  • при перимаканні швидкостей неможна щоб ланцюг виявился на двух великих зірочках. Так вона швидше зноситься;
  • при перемиканні заборонено, щоб ланцюг був на двух маленьких зірочках. Вона може зпасти з зірок.

    Підбір велосипеда

  • ціль катання;
  • вага клієнта, поверхня катання.

    • якщо вага більше 100кг та планує кататися по кочкам, то обираємо усилену раму, двойні обода, пружино-масляну або повітряно-масляну вилку.

    якщо вага до 100кг та рівна дорога, то обираємо звичайну раму, будьякі обода, будьякий тип амортизаці вилки.

  • погода;

    • будьяка погода - дискові гальма. Сонячна та сіха погода - будьякі гальма.

  • частота використання та інтенсивність

    • при інтенсивному катанні - веліка кількість швидкостей, система перемикання швидкостей рівня Deore і віще.

    при не частому катанню по місту - будьяка кількість швидкостей, будьяка система перемикання.
