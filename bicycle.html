Велосипеди

Підберається в залежності від того де планується використування. Це може бути велосипед для швидкого приїзду на роботу, для походу, кататися по парку та відпочивати. Чи можливо для поїздок з друзями на пікнік.

    • Горний

    Їзда за містом по бездоріжжю.

    рама стальна, алюміньова або карбон
    вилка амортизуюча
    гальма v-brake або дискові
    Передачі великий діапазон

    Великий діаметр колес, для того щоб легше проходити нерівності. Амортизаційна вилка, для того щоб їздити по камінню та кочкам. Великий діапазон передач для того щоб було легше підніматися до гори.

    Міський

    Для їзди в місті та по асфальту.

    рама стальна або алюміньова
    вилка жортска, без амортизаторів
    гальма v-brake
    Передачі відсутні або небагато

    Широке комфортне сідло. Часто доповнюється корзинкою або багажником. Широке та високе кермо для прямої посадки.

    Електричний

    Бувають для міста та для бездоріжжя.

    рама стальна або алюміньова
    вилка жортска або з амортизатором
    гальма v-brake або дискові

    Двигун надає підтримку до 25 км/ч. Вага більш висока. Відрізняються часом заряду та дальністтю поїздки.

    Складний

    Для поїздок у місті

    рама стальна або алюміньова
    вилка жортска
    гальма v-brake
    Передачі є невелика кількість

    Займають меньше місця, простіше перевозити. Чим швижше і простіше складаєтся ти дорожче ціна.

    Шосейний

    Для швидкостної їзди по дуже рівної поверхні

    рама алюміньова
    вилка жортска
    гальма v-brake
    Передачі велика кількість

    Спортивна посадка та вигнуте кермо. Дуже вузькі колеса с гладкими покришками які знижують тертя.

    BMX

    Для трюків

    рама стальна
    вилка жортска
    гальма v-brake
    Передачі відсутні

    Тільки для "флета" та "стріта". Кермо крутиться на 360. Спеціальні труби на осях "пеги". Пеги потрібні для виконання різноманітних трюків.
