Особливості стопи

Пронация

Елемент природнього руху людського тіла. Спосіб постановки зовнішньої частини стопи для розпроділу навантаження. Знання типу прониції клієнта допоможе підібрати зручне взуття для комфортного життя.

Гіпопронація

Характерне надмірне завалювання назовню, високий свод стопи. Як визначити - знос взуття з завнішньої сторони, невеличкий нахил підошви зовні при зовнішньому огляді.

Яке обираємо взуття:

Стакан с жорсткою п'яткою. Збільшена амортизація по зовнішній стороні підошви. Обираємо анатомічну устілку.

Нейтральна пронація

Як бачимо з назви, це звичайний свод стопи. Для цього типу присущій знос взуття по зовнішньому краю п'ятки та по внутрішньому краю носка.

Спецпідтримка не потрібна. Амортизація не буде зайвою.

Гиперпронація

Характерне завалювання всередину, низький свод стопи та плоскостопість. Зношення по внутрішньому краю від п'ятки до великого пальця. Невеличкий нахил підошви всередину.

Потрібна стабілізація з внутрішнього боку. Жорсткі вставки у середню частину підошви. Амортизація.
